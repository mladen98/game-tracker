\# Security Policy



This is a learning/portfolio project.



\## Reporting a vulnerability

If you discover a security issue, please open a private message to the repository owner or create an issue with minimal details.

Do not publish sensitive information (tokens, secrets, personal data) in issues or pull requests.



\## Notes

\- Never commit secrets (API keys, tokens, passwords).

\- Use environment variables / config files that are excluded via `.gitignore`.
